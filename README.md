# Zeta

Zeta é uma plataforma para transformar uma conversa sobre o negócio em produto digital pronto para uso: site, CRM e sistema web personalizado.

## Como o Zeta vai funcionar

O Zeta começa como uma experiência simples: o cliente escolhe o que precisa, informa o contexto da empresa e recebe um caminho claro para sair do improviso digital.

Na prática, ele será dividido em três produtos:

- **Zeta Site**: cria presença digital, páginas institucionais e páginas de venda com copy clara, estrutura profissional e base de SEO.
- **Zeta CRM**: organiza clientes, leads, oportunidades e rotina comercial sem depender de planilhas soltas.
- **Zeta OS**: transforma processos internos, agenda, atendimento e operação em um sistema web sob medida.

Também existe o plano **Tudo junto**, que combina os três produtos em um pacote único para empresas que querem presença, vendas e operação conectadas desde o começo.

## Fluxo previsto

1. O cliente entra no site.
2. Escolhe um produto individual ou o pacote completo.
3. Cria uma conta ou entra com uma conta existente.
4. Responde perguntas sobre a empresa, processo e objetivo.
5. O Zeta organiza esse contexto em briefing, escopo e próximos passos.
6. A entrega evolui para site, CRM ou sistema web, dependendo do plano escolhido.

## Rotas

- `/`: apresentação do Zeta.
- `/produtos`: produtos, mensal/anual e pacote completo.
- `/entrar`: tela visual de cadastro/login.
